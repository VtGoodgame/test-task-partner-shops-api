## 🚀 Технологии

- **Python 3.12+** - основной язык программирования
- **FastAPI** - веб-фреймворк для создания API
- **SQLAlchemy 2.0+** - ORM для работы с базами данных
- **aiosqlite** - асинхронный драйвер для SQLite
- **Uvicorn** - ASGI сервер для запуска приложения
- **Make** - автоматизация задач

### Ключевые компоненты:

1. **Database класс** (`db/database.py`):
   - Асинхронная работа с SQLite через `aiosqlite`
   - Контекстный менеджер для управления соединениями
   - Безопасное выполнение SQL запросов с защитой от инъекций
   - Метод для создания таблиц из SQLAlchemy моделей
   - Специализированный метод для получения магазинов с ближайшей доставкой

2. **Модели данных** (`db/models.py`):
   - `PartnerStores` - магазины-партнеры
   - `DeliverySlots` - интервалы доставки
   - Связь один-ко-многим между магазинами и слотами доставки

3. **API слой** (`app/main.py`):
   - FastAPI приложение с эндпоинтами
   - Обработка ошибок и валидация

## API Эндпоинты

### GET `/api/v1/shops/with-delivery-by-hour`

Получение списка активных магазинов с ближайшим интервалом доставки к указанному часу.

**Параметры запроса:**

| Параметр | Тип | Обязательный | Описание |
|----------|-----|--------------|----------|
| `hour` | integer | Да | Час дня (0-23), для которого нужно найти ближайшие интервалы доставки |

**Успешный ответ (200 OK):**

```json
[
  {
    "id": 1,
    "logo_url": "/src/images/lenta.png",
    "name": "Лента",
    "delivery": 1,
    "official_website": "https://lenta.com",
    "is_active": 1,
    "nearest_delivery_start": "10:00",
    "nearest_delivery_end": "11:00",
    "hour_difference": 0,
    "has_delivery": 1
  }
]
```
### Поля ответа:
| Поле | Тип | Описание |
|------|-----|----------|
|id|integer|Уникальный идентификатор магазина|
|logo_url|string|URL логотипа магазина|
|name|string|Название магазина|
|delivery|boolean|Доступность доставки (1 - есть, 0 - нет)|
|official_website|string|Официальный сайт магазина|
|is_active|boolean|Статус активности магазина (1 - активен, 0 - неактивен)|
|nearest_delivery_start|string|Начало ближайшего интервала доставки (формат HH:MM)|
|nearest_delivery_end|string|Конец ближайшего интервала доставки (формат HH:MM)|
|hour_difference|integer|Разница в часах между запрошенным часом и началом доставки|
|has_delivery|boolean|Наличие интервала доставки (1 - есть, 0 - нет)|

## Ошибка (404 Not Found):
```json
{
  "detail": "Активные магазины не найдены"
}
```
